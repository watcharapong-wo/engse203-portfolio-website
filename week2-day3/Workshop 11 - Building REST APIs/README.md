# Workshop 11: Building REST APIs

## รายละเอียด
ฝึกสร้าง RESTful API สำหรับจัดการข้อมูล (เช่น Product Management)

## โครงสร้างโปรเจกต์
- routes/ : ไฟล์ route ของ API
- controllers/ : (ถ้ามี) สำหรับแยก logic
- index.js : จุดเริ่มต้นของแอป

## วิธีใช้งาน
1. ติดตั้ง dependencies
   ```sh
   npm install
   ```
2. รันเซิร์ฟเวอร์
   ```sh
   node index.js
   ```
