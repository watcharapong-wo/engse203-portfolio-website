# Workshop 10: Express.js Basics

## รายละเอียด
ฝึกสร้าง REST API เบื้องต้นด้วย Express.js

## โครงสร้างโปรเจกต์
- routes/ : ไฟล์ route ของ Express
- app.js หรือ index.js : จุดเริ่มต้นของแอป

## วิธีใช้งาน
1. ติดตั้ง dependencies
   ```sh
   npm install
   ```
2. รันเซิร์ฟเวอร์
   ```sh
   node index.js
   ```
