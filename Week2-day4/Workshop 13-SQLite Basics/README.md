# Workshop 13: SQLite Basics

## รายละเอียด
ฝึกใช้งานฐานข้อมูล SQLite เบื้องต้น เช่น การสร้างตาราง เพิ่ม ลบ แก้ไข ค้นหา

## โครงสร้างโปรเจกต์
- database/ : ไฟล์ฐานข้อมูลและสคริปต์
- main.js หรือ index.js : จุดเริ่มต้นของแอป

## วิธีใช้งาน
1. ติดตั้ง dependencies (ถ้ามี)
2. รันโปรแกรม
   ```sh
   node main.js
   ```
