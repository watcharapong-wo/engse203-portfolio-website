# Workshop 14: Database Integration with Node.js

## รายละเอียด
ฝึกเชื่อมต่อ Node.js กับฐานข้อมูล (เช่น SQLite, MongoDB)

## โครงสร้างโปรเจกต์
- routes/ : ไฟล์ route
- database/ : ไฟล์ฐานข้อมูล
- index.js : จุดเริ่มต้นของแอป

## วิธีใช้งาน
1. ติดตั้ง dependencies
   ```sh
   npm install
   ```
2. รันเซิร์ฟเวอร์
   ```sh
   node index.js
   ```
